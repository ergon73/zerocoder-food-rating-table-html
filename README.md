# Рейтинг любимых и нелюбимых блюд

Учебный проект по HTML для курса Python по основам веб-разработке.

## Описание

Простая HTML-страница с таблицей рейтинга блюд. Таблица содержит:
- Название блюда
- Оценку от 1 до 10
- Ссылку на рецепт
- Фото блюда

## Технологии

- HTML5
- Только встроенные HTML-атрибуты (без CSS и JavaScript)

## Структура проекта

```
.
├── index.html    # Основная страница с таблицей
└── README.md     # Этот файл
```

## Как открыть

Просто откройте файл `index.html` в любом браузере.

## Требования

Никаких дополнительных зависимостей не требуется. Просто HTML-файл.
